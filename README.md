# Padrões de Projeto - Grupo 7

## Descrição

Este repositório reúne exemplos práticos de implementação de padrões de projeto apresentados em sala de aula.

Os exemplos foram desenvolvidos com foco educacional, demonstrando tanto problemas comuns de arquitetura quanto soluções utilizando Design Patterns.

Cada padrão foi organizado em sua própria pasta, contendo:
- código-fonte;
- exemplos de utilização;
- README específico;
- explicação do funcionamento.

---

# Padrões Implementados

## Adapter

Demonstra como adaptar interfaces incompatíveis utilizando um intermediário responsável pela tradução de chamadas.

### Objetivo
Reduzir acoplamento entre o sistema principal e bibliotecas externas.

### Tecnologias
- Python
- Programação Orientada a Objetos

---

## Builder

Demonstra a construção gradual de objetos complexos utilizando encadeamento de métodos.

### Objetivo
Separar a lógica de construção da representação final do objeto.

### Tecnologias
- Python
- Programação Orientada a Objetos

---

## Template Method

Demonstra a definição de um algoritmo base reutilizável, permitindo que subclasses implementem etapas específicas.

O projeto apresenta:
- implementação sem o padrão;
- implementação utilizando Template Method.

### Objetivo
Padronizar algoritmos e reduzir duplicação de código.

### Tecnologias
- Java
- Programação Orientada a Objetos
